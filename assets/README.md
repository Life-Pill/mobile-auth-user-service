# LifePill Assets

This folder contains brand assets for the LifePill application.

## Logo

Place the `lifepill-logo.png` file in this directory for email templates.

### Logo Requirements:
- **Format:** PNG with transparent background
- **Size:** 200x200 pixels (recommended)
- **Usage:** Email headers, notifications

### Logo URL
The logo is referenced in email templates from:
```
https://raw.githubusercontent.com/Life-Pill/mobile-auth-user-service/main/assets/lifepill-logo.png
```

## Brand Colors

| Color Name | Hex Code | Usage |
|------------|----------|-------|
| Primary Cyan | `#00B4D8` | Main brand color |
| Primary Blue | `#0077B6` | Gradient end |
| Dark Blue | `#1E3A5F` | "PILL" text |
| Teal Start | `#4FD1C5` | Alternative gradient |
